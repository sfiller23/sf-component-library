# Pad

The Pad Component is a wrapper for creating spaces around the child or children components using the theme schema for spaces, Keeping the padding one line shortcut logic.

#### Example

<Canvas of={Pad.Different} />

## Usage

```ts
import {Pad} from "@cman430/sf-component-library";

const Example = () => {
  return (
    <div>
      <Pad space="s m l xl">
        <div>Child</div>
      </Pad>
    </div>
  );
};

export default Example;
```

#### Arguments

- **space** _`string`_ - A string that represents the padding value according to the theme space schema

#### Schema

```ts
export const spaceSchema: ISpaceSchema = {
  xs: "0.125rem",
  s: "0.25rem",
  m: "0.5rem",
  l: "1rem",
  xl: "2rem",
  xxl: "4rem",
  none: "0rem",
};
```
